# App Orçamento Pessoal

É um App de Orçamento Pessoal, que tem como objetivo o cadastro, consulta e remoção de despesas. Foi utilizado o Paradigma de Orientação a Objetos.

## Tecnologias

- HTML
- CSS
- JS
- Bootstrap
- ### Banco de Dados
  - LocalStorage
